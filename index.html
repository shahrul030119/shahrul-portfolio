My Highlighted Works

Final Year Project

This mobile-based application is designed to support rubber tappers by providing centralized access to essential data. It utilizes a Python web scraping script to retrieve real-time rubber price data from the Malaysia Rubber Board and weather updates in Kelantan from the MetMalaysia website. Key features include push notifications, a productivity graph, latex input tracking, and tapping day logs. The system also includes a dedicated admin web portal, enabling administrators to manage user data, post announcements, and update information efficiently.
